🛡️ MedGuard: AI Hallucination Detection for Medical Chatbots

MedGuard is a full-stack application that improves the safety of medical AI by detecting hallucinations and risky responses using a Dual-Layer Verification System.

🚀 Features

🧠 AI chatbot for medical queries
🔍 Rule-based risk detection (dosage, surgery, etc.)
📚 Knowledge validation using verified dataset
📊 Confidence score (0–100) for each response
🟢🟡🔴 Safety indicators (Safe / Uncertain / High Risk)

🛠️ Tech Stack

Frontend: React (Vite), Framer Motion
Backend: FastAPI (Python)
AI Logic: Rule Engine + Dataset Validation.
